What is eEvent Booking System?

eEvent Booking System is a powerful Direct-to-Consumer (D2C) event management and ticketing platform built on the secure and scalable MERN (MongoDB, Express, React, Node.js) stack. It is tailored for individual event organizers and agencies who want to manage and sell their events directly to attendees without relying on third-party platforms.

The system includes a modern, rich admin panel that allows full control over every aspect of the website‚Äîevent listings, ticketing, content, users, and more. All website content is dynamically managed from the admin, making it easy for non-technical users to update and customize the platform to match their brand and goals.

With eEvent, organizers can effortlessly create beautiful event pages, manage attendee registrations, filter events by country, state, or city, publish engaging blogs, and access in-depth sales reports. Whether you're hosting workshops, conferences, concerts, or local experiences, eEvent empowers D2C organizers with the tools they need to build, promote, and sell events independently‚Äîwhile maintaining complete ownership of their audience and data.

How to Setup "Find E-Event" API on Your Local Server?

1. Install Required Software

  1. Operating System: Windows, Linux.
  2. IDE: VSCode
  3. Basic knowledge of React, Node, express and MongoDB.
  4. Install NodeJS

    5. Note: Download and install NodeJS from the official website: https://nodejs.org/

  5. Install MongoDB

    6. Note: Download and install MongoDB Community Edition:https://www.mongodb.com/try/download/community And Follow the steps: https://www.geeksforgeeks.org/how-to-install-mongodb-on-windows/

    OR

    Download MongoDB Database Tools

    Go to: MongoDB Tools Download

    1. Choose your operating system: Windows.
    2. Download and install the MongoDB Database Tools.

    Add MongoDB Tools to PATH

    1. Press Win + R, type sysdm.cpl, and hit Enter.
    2. Go to Advanced ‚Üí Environment Variables.
    3. Find Path under System Variables and click Edit.
    4. Add the MongoDB bin directory (e.g., C:\Program Files\MongoDB\Tools\bin).
    5. Click OK to save the changes.

2. Create database in MongoDB Atlas

  • If you have downloaded the source code from CodeCanyon, unzip the folder.
  • Open a terminal in the root folder.
  1. Login Or SignUp at MongoDB Atlas

  2. Create a new project

  3. Set project name and create project

  4. Create cluster

  5. Choose Plan > Set cluster name and copy > Select cloud provider and region > Click on "Create Deployment" button

  6. Copy username and password > Click on "Create database Use' button > Click on "Choose a connection method" button > Click on "Close" button

  7. Go to Network Access option from the left side menu > Click on "+ADD IP ADDRESS" button > Click on "Allow access from anywhere" button > Click on "Confirm" button

  8. To import the database > Run VSCode > Import project folder > run the following command in the project root terminal. 
    9. mongorestore --uri "mongodb+srv://<username>:<db_password>@<clustername>.mongodb.net" <path>

    OR

    Go to MongoDB Atlas > Click on "Connect" button at Clusters section > Click on "Drivers" > Copy Connection string > Paste it inbetween "" comma with above command line

    Replace <username> with your admin username, <db_password> with your database password, and <clustername> with your cluter name, and <path> with the folder name containing the database.

    for more information visit this link https://www.mongodb.com/docs/atlas/import/mongoimport/

4. Install Dependencies

To run api locally

cd API
npm install

5. Set Up Environment Variables

cp .env.example .env

Go to API folder > Edit the .env file and set database credentials and other variables

DB_CONNECTION=<CONNECTION_STRING>
ACCESS_TOKEN_SECRET=<ACCESS_TOKEN_SECRET>
REFRESH_TOKEN_SECRET=<REFRESH_TOKEN_SECRET>
PORT=<PORT>

  • Go to MongoDB Atlas and click the "Connect" button in the Clusters section.

    Select the "Drivers" option, copy the connection string, and replace <CONNECTION_STRING> with it.

    Ensure you replace <db_password> with your actual database password.

  • Run following command Twice at terminal so you can get <ACCESS_TOKEN_SECRET> and <REFRESH_TOKEN_SECRET>

    node -e "console.log(require('crypto').randomBytes(64).toString('hex'))"

  • Set the value of PORT=5600 in the .env file:

    PORT=5600

6. Run the Development Server

npm start

Access the API at http://localhost:5600/api

Deployment

Deploy API on Vercel Server

  1. Login Or SignUp at Vecel

  2. Go to API terminal and Install Vercel CLI by following command
    3. npm install -g vercel
  3. Login to Vercel by following command
    4. vercel login
  4. Select suitable method for login which you have used while signup
  5. Deploy to production
    6. vercel --prod
  6. Read questions and answer them accordingly.

  7. Go to vercel website dashboard (1) > Select created project (2)

  8. Click on "Settings" tab from top menu (1) > Select "Environment Variables" from left menu (2) > Click on "Import.env" button (3) > Import .env file from project folder > Save (4)

  9. You will get Redeploy popup > Click on "Redeploy" button and finish process.
  10. API deployment completed.
  11. Go to dashboard and find API URL > Copy and Save it.

Deploy Admin on Vercel Server

  1. Go to Admin terminal and Run following command
    2. vercel --prod
  2. Read questions and answer them accordingly.

  3. Go to vercel website dashboard (1) > Select created project (2)

  4. Click on "Settings" tab from top menu (1) > Select "Environment Variables" from left menu (2) > Add Key "REACT_APP_API_URL" (3) > Paste API URL at Value (4) Click on "Save" button (5)

  5. You will get Redeploy popup > Click on "Redeploy" button and finish process.
  6. Admin deployment completed.
  7. Go to dashboard and Click on Admin URL

Deploy Website on Vercel Server

  1. Go to Website terminal and Run following command
    2. vercel --prod
  2. Read questions and answer them accordingly.

  3. Go to vercel website dashboard (1) > Select created project (2)

  4. Click on "Settings" tab from top menu (1) > Select "Environment Variables" from left menu (2) > Add Key "REACT_APP_API_URL" (3) > Paste API URL at Value (4) Click on "Save" button (5)

  5. You will get Redeploy popup > Click on "Redeploy" button and finish process.
  6. Website deployment completed.
  7. Go to dashboard and Click on Website URL
